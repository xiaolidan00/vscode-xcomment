# vscode-xcomment 自动添加注释

支持文件js,ts,vue

- 全文档给函数添加注释
- 全文档给参数属性等添加注释
-

https://astexplorer.net/

JS JSX AST
https://github.com/acornjs/acorn

VUE AST
@vue/compiler-dom

TS TSX AST ts-morph

- 右击选择`XComment:All`全文档添加注释
- 选中内容，右击`XComment:Add`添加注释
- 快捷键：`Alt+/`添加注释
