# vscode-xcomment 自动添加注释

支持文件js,ts,vue

选中代码或打开文档（未选择文本默认全文档）

- 右击选择`xComment:Method`或使用快捷键`Alt+A+/`给函数方法添加注释
- 选中内容，右击`XComment:Params`或使用快捷键`Alt+S+/`给参数属性等添加注释
